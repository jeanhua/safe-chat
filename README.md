## 使用教程🔎
客户端：下载release打开即可  
服务器：服务器端代码在server文件夹下，运行server.py即可  


## 注意事项🧨
**AES密钥必须为16位整数，输入$代表随机生成**  
**本项目未对中间人攻击和社会工程学攻击设计防范，对于绝密场景谨慎使用**
